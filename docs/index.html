#BlockPartyKit

Bring your laptop instead of a dish to pass.

How the party works

Host

One machine runs one command. It creates a private mesh and prints an invite code. Write it on the whiteboard next to the snack list.

./scripts/host.sh

Join

Everyone runs one command. Each laptop adds its memory and compute to the cluster. Watch the machines find each other. Cheer.

./scripts/join.sh <CODE>

Use

Start goose. Now the whole room talks to the same local AI, and the room itself is the brain. Let the youngest kiddo ask the first question.

./scripts/librarian.sh

Built on

Mesh LLM

Pools GPUs and memory across laptops into one local inference API. Can find its peers over plain local Wi-Fi, so it keeps working when the internet doesn't.

goose

The open-source agent originally developed at Block, now part of the Linux Foundation ecosystem, with Spiral's new GDK for building on top.

The kit is a thin layer over existing open software. The experiment isn't the technology. It's getting a room full of people to run it together.
